#  Clima App ☀️

#PROBAR: https://clima-fedelei.netlify.app/  

✅ Buscador de clima en tiempo real desarrollado con ReactJS, que permite a los usuarios consultar las condiciones meteorológicas de cualquier ciudad. La interfaz es simple, rápida y fácil de usar, pero a la vez moderna con animaciones, pensada para ofrecer una experiencia intuitiva.


✅ Además, el manejo de estado y las llamadas a la API están optimizados para brindar resultados inmediatos y precisos.
